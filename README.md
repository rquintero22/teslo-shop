# TesloDB Api

1. Clonar proyecto
2. ```yarn install```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las variables de entorno
5. Iniciar la base de datos
```docker-compose up -d```

6. Ejecutar el Seed 
```
http://localhost:3000/api/seed
```

7. Iniciar ```yarn start:dev```
